# GestiStock

Application de gestion de stock développée avec Symfony et JavaScript.

## Fonctionnalités

- CRUD Catégories
- CRUD Produits
- Gestion des mouvements (entrée / sortie)
- Mise à jour automatique du stock
- Historique des mouvements

## Stack

- Symfony (API REST)
- JavaScript Vanilla (frontend)
- MySQL
